# Situated Views of Rio de Janeiro

> Repository for tracking technical issues and generating static files from kmls.

## Generating files for distribution

```bash
# install dependencies
$ npm install
$ pipenv install

# kml parser
$ npm run g

```

## Folder structure

```
situated-views
│   index.js
│   package.json
│   Pipfile.json
│   README.md
│

│
```
